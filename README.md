# Brimble Mini-Deployment Platform

A simplified deployment platform similar to a mini Heroku or Vercel, built as a fullstack/infra engineering take-home project.

## Key Resources
- [Project Scope](./project-scope.txt): The core requirements and goals.
- [Project Phases](./directives/project_phases.md): The step-by-step roadmap for implementation.

## Tech Stack
- **Frontend:** Vite, TanStack Router, TanStack Query
- **Backend:** Node.js, TypeScript, Express/Fastify
- **Database:** SQLite
- **Infrastructure:** Docker, Docker Compose, Caddy, Railpack

## Getting Started
The entire system is designed to run with a single command:
```bash
docker compose up
```
